# プロダクト概要

Help Naviは、Next.js + Mastra + Claude APIを統合したAIチャットアシスタントWebアプリケーションである。
ユーザーとの対話型チャットインターフェースを通じて、AIエージェントがファイル管理を含む様々なタスクを支援する。
会話セッションの永続化により、過去のやり取りを保存・参照できる。

## コア機能

- **ストリーミングチャット**: useChat()フックによるリアルタイムSSEストリーミング対話。Streamdownによるストリーミング対応Markdownレンダリング
- **会話管理**: PostgreSQLによる会話セッション・メッセージの永続化。サイドバーから会話の一覧表示・選択・タイトル編集・削除が可能。初回メッセージ送信時にLLMによるタイトル自動生成
- **S3ファイル操作**: エージェントツールとしてのファイル一覧取得・読み取り・アップロード
- **ローカルインフラ**: Docker ComposeによるMinIO + PostgreSQLの自動セットアップ

## 対象ユースケース

- AIエージェントとの自然言語によるチャットインタラクション
- 会話履歴の保存と過去のやり取りの参照・管理
- エージェント経由でのS3互換ストレージのファイル管理
- Mastraフレームワークを活用したエージェント開発の学習・プロトタイピング

## 提供価値

- Next.js + Mastra + Claude APIの統合パターンを提供する開発テンプレート
- ストリーミングチャットによるリアルタイムなエージェント対話を実装済み
- 会話セッション管理と永続化による継続的なコンテキスト保持
- Docker ComposeによるMinIO + PostgreSQLの自動起動で、外部サービス依存なくローカル開発可能
- 日本語対応のUIとエージェント指示
